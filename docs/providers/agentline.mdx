[AgentLine](https://agentline.cloud) gives AI agents real phone numbers for making outbound voice calls and reading inbound SMS messages. Your LangChain agents can talk to real people on the phone and receive text messages from real phone numbers.

## Installation and setup

Install the Python package:

```bash
pip install langchain-agentline
```

Set your API key:

```bash
export AGENTLINE_API_KEY="sk_live_your_key"
```

## Tools

See the [AgentLine Toolkit](/oss/integrations/tools/agentline) guide for details on all available tools:

- `agentline_make_call` — Make outbound phone calls
- `agentline_get_calls` — Check call status and get transcripts
- `agentline_get_messages` — Read inbound SMS messages
- `agentline_list_agents` — List configured agents and phone numbers
